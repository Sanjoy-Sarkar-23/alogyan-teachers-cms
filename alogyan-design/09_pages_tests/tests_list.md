# Tests List

**References:** Zoho (event lists), Notion (calendars).

## 1. Layout
Grid based dashboard splitting "Upcoming Tests" and "Past Assessments".

## 2. Components
- **Test Card:** Subject, Date, Status (Upcoming, Needs Grading, Graded).

## 3. Tables
Used inside specific cards if "List View" is toggled, showing columns for Name, Date, Avg Score.

## 4. Forms
Filters for Subject or Batch.

## 5. Buttons
Primary Header Button: `+ Schedule Test`.

## 6. UX Flow
Instantly acts as a calendar overlay to show teachers what assessments are coming up next without leaving the main module layout.

## 7. Mobile Layout
Vertically stacked cards.

## 8. Empty States
"No tests scheduled. Keep students engaged with regular assessments!"

## 9. Success States
Hover states elevate the cards (`box-shadow` depth increases).

## 10. Error States
"Could not load upcoming tests. Try again."
